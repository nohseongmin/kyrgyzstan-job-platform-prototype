# Kyrgyzstan Job Platform Prototype

A frontend prototype for a Kyrgyzstan job portal (Работа.kg) similar to Albamon/Alba Cheonguk, built with HTML, CSS, and vanilla JavaScript.

## Features
- **Job Listing (index.html):** Displays available job postings. Uses `localStorage` to fetch dynamically added jobs.
- **Post a Job (post-job.html):** A form for employers to create job listings. Simulates a backend by saving postings to `localStorage`.
- **Apply for a Job (apply-job.html):** A form for job seekers to apply. Contains simulated application submission.
- **Dynamic Interactions:** Employs event delegation to handle interactions (likes and applying) on dynamically generated job listings seamlessly.

## Setup
There are no backend requirements or build steps. Simply open `index.html` in your web browser to start using the prototype.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+ Vanilla JS)
- Ionicons for iconography
